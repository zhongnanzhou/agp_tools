# AGP - Automated Graphic Processing Tools

> **当前版本：v0.0.1** | PySide6 桌面图形处理工具

一个集成化的图片处理工具，基于 PySide6 构建，支持角度检测、角度校正、图片切分、图片压缩等功能。提供三栏式工作台界面，包含目录缩略图预览、图片编辑预览、功能面板和控制台输出。

## 技术栈

| 类别 | 技术 |
|------|------|
| GUI 框架 | PySide6 >= 6.5.0 |
| 图像处理 | OpenCV >= 4.8.0、Pillow >= 10.0.0 |
| 数值计算 | NumPy >= 1.24.0 |
| Python | 3.10+ |

### 核心设计

- **观察者模式**：Qt 信号槽实现面板与主窗口的解耦通信
- **统一日志系统**：基于 Python logging，支持 console / file / qt 多 handler 独立级别输出
- **懒加载策略**：目录缩略图使用 QImageReader 直接解码到目标尺寸 + 可见区域估算加载
- **内存处理**：图片操作在内存中进行，通过 PIL Image 对象传递

## 安装与运行

### 安装依赖

```bash
cd tools/agp
pip install -r requirements.txt
```

### 运行程序

```bash
# 从 tools 目录运行
cd tools
python -m agp.main
```

### 操作流程

1. **加载图片**：通过菜单「文件 → 打开图片」或拖拽图片到预览区域
2. **加载目录**：通过菜单「文件 → 打开目录」或拖拽目录到左侧栏
3. **选择图片**：点击左侧缩略图切换当前预览图片
4. **执行功能**：点击右侧功能按钮执行对应处理
5. **查看结果**：底部控制台显示日志，执行结果标签页显示处理结果
6. **保存图片**：通过菜单「文件 → 保存」或「另存为」

#### ⚡ 性能优化
- 目录缩略图使用 QImageReader 直接解码到目标尺寸，避免全量加载原图（内存降低 80%+）
- 可见区域估算加载（indexAt + 行列估算），替代全量遍历（滚动时 CPU 从 O(n) 降到 O(可见项)）
- 缩略图加载日志降级为 DEBUG，减少控制台刷屏

#### 📦 工具支撑
- 统一日志系统（支持 console / file / qt 多 handler）
- Qt 日志桥接处理器
