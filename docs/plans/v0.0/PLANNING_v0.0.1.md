# PLANNING_v0.0.1

> 版本计划：初始版本 — 三栏框架 + 四核心功能

---

## 1. Context（背景）

**所属里程碑**：v0.0

**版本目标**：搭建 PySide6 三栏式主窗口框架，集成四项核心图片处理功能

**关联需求**：
- 需求来源：从 `image_crop/` 原型项目迁移为集成化桌面工具

---

## 2. Goals（目标）

### 核心目标
1. 搭建 PySide6 三栏式主窗口框架（目录预览 | 图片预览+控制台 | 功能面板）
2. 集成四项核心图片处理功能（角度检测/校正、图片切分、压缩）
3. 实现缩略图懒加载性能优化

### 验收标准
- [x] `python -m agp.main` 可启动主界面
- [x] 四功能按钮可完成一次最小闭环执行
- [x] 缩略图懒加载正常工作，内存使用合理
