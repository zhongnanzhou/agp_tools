## 2. Goals（目标）

### 核心目标
1. 拖拽图片文件（非目录）到目录预览区，图片追加到缩略图列表
2. 拖入单图后点击缩略图可正常加载到预览区

### 验收标准
- [ ] 拖拽图片文件（非目录）到目录预览区，不报错，图片出现在缩略图列表中
- [ ] 拖拽图片后，点击缩略图可正常加载到预览区（通过 EventBus `image_selected`）
- [ ] 拖拽目录行为不变，与现有逻辑一致
- [ ] 提示文字更新，反映两种拖拽模式
