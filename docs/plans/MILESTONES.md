## v0.0 基础框架与核心功能

**状态**：✅ 已完成

**目标**：搭建 PySide6 三栏式工作台框架，实现四项核心图片处理功能，完成组件通信解耦重构

**完成标准**：
- [x] 主窗口三栏布局（目录预览 | 图片预览+控制台 | 功能面板）
- [x] 四项核心功能可闭环执行（角度检测/校正、图片切分、压缩）
- [x] 组件通信解耦（EventBus 事件总线 + Widget 移除冗余引用）
- [x] 统一包导入路径，`python -m agp.main` 可稳定启动
